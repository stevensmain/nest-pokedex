# Run on development
1. Clone github repository
2. Run
```
 npm install
```
3. If you dont have Nest CLI installed, run:
```
npm i -g @nestjs/cli
```
4. Run database
```
docker-compose up -d
```

## Stack
 * NestJs
 * MongoDB
